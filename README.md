# 🍔 Swiggy Sales Analysis — SQL Project

## Overview
End-to-end SQL Server project analysing Swiggy food delivery data
across Indian cities, restaurants, and cuisines.

## What This Project Covers
- Data Validation & Cleaning (Null checks, blank strings, duplicates)
- Star Schema Design (4 dimension tables + 1 fact table)
- KPI Development (Total Orders, Revenue, Avg Price, Avg Rating)
- Business Analysis (Monthly trends, Top cities, Cuisine performance, Spending insights)

## Tools Used
- SQL Server (T-SQL)
- Microsoft Excel (raw data source)
